S3

Usage Patterns

  • store and distribute static web content and media
    • delivered directly from Amazon S3 with a unique HTTP URL.
    • serve as an origin store for a content delivery network (CDN), such as Amazon CloudFront
  • host entire static websites
  • data store for computation and large-scale analytics
    • such as financial transaction analysis, clickstream analytics, and media transcoding
    • Horizontal scalability
    • access your data from multiple computing nodes concurrently
    • without being constrained by a single connection
  • backup and archiving of critical data
    • You can easily move cold data to Amazon S3 Glacier using lifecycle management rules on data stored in Amazon S3
    • EBS snapshot
    • For very long-term backups of MariaDB, MySQL, and PostgreSQL data, we recommend exporting snapshot data to Amazon S3

Performance

multipart upload

To improve the upload performance of large objects (typically over 100 MB), Amazon S3 offers a multipart upload command to upload a single object as a set of parts. After all parts of your object are uploaded, Amazon S3 assembles these parts and creates the object.

S3 Transfer Accelerator

  • Use Amazon CloudFront globally distributed edge locations to route traffic to your Amazon S3 bucket over an Amazon-optimized network path.
  • To get started with Amazon S3 Transfer Acceleration:
    • you first must enable it on an Amazon S3 bucket.
    • Then modify your Amazon S3 PUT and GET requests to use the s3-accelerate endpoint domain name (<bucketname>.s3-accelerate.amazonaws.com).
  • The Amazon S3 bucket can still be accessed using the regular endpoint.

Security (All applies on bucket)

  • Versioning
  • MFA
  • Access Logging
  • Security at rest
    • Server-side Encryption
      • SSE-KMS (Envelope encryption, AWS KMS and you manage the keys together)
      • SSE-C (Customer provided key (you manage the keys yourself))
      • SSE-AES (S3 handles the key, uses the AES-256 alg)
    • Client-side Encryption
  • Security in transit
    • Client-side Encryption
    • SSL
