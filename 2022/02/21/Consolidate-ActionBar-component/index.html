Consolidate ActionBar component
Post：2022-02-21 10:28:38 Tags：/ React / Visit：

RFC

Situation

There are two different types of ActionBar components, ActionBar which is an inline action bar, but can be overwritten to look like a page action bar, and ActionBarWithHelp which is always a page action bar, but can only be used with help links and actions.
These components provide a confusing interface for developers which often causes the misuse of each component.
For example, inline action bars are often used as page action bars.

Requirements

  1. The new action bar should be usable as either a page or inline action bar.
  2. Minimal impact rollout - this should be rolled out with the lowest impact on consumers as possible.
  3. API improvements -The current API is confusing with hard to use props, we should see if we can make any improvements as part of this work.

Proposal: a new compositional component

Composite structure

Reading: Avoid soul-crushing components

This pattern helps to break functionality out into smaller components and makes it easier to add functionality, like other sections, in the future.

Variant prop

We can consolidate these into one component by adding a variant prop to the new ActionBar component. The possible values will be either page or inline.

Minimal impact rollout

To ensure this has minimal impact on users of the ActionBar components we should…

  • Deprecate the ActionBarWithHelp component. - This will mean current users of this component can continue using it as it was but will be encouraged to migrate to the new component.
  • Make the variant prop optional and default to inline this will mean existing users of the ActionBar component will be unaffected. This is because that component is already a component which just wraps its children in a styled div.

Plan

  1. Create compound component
    • Write new compound component
    • Update ActionBar storybook
    • Adapt existing ActionBar tests and write new ones
    • Check existing uses of ActionBar are unaffected
  2. Release - minor version
  3. Deprecate ActionBarWithHelp
    • Deprecate
  4. Release - major version
